## Development Setup Local

1. Clone repository
```
git clone https://github.com/AlexGalhardo/todo-api-clean-architecture.alexgalhardo.com
```

2. Enter repository
```
cd todo-api-clean-architecture.alexgalhardo.com/
```

3. Run setup.sh
```
chmod +x ./setup.sh && ./setup.sh
```

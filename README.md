## Introduction

* A simple MVC full stack web application using NodeJS.

## Development Setup Local

1. Clone repository
```
git clone git@github.com:AlexGalhardo/galhardo-app.git
```

2. Enter repository
```
cd app.alexgalhardo.com/
```

3. Run setup.sh
```
chmod +x ./setup.sh && ./setup.sh
```
